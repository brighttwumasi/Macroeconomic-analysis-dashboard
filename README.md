# 📊 Macroeconomic Indicators Analysis Dashboard

An interactive Power BI dashboard developed to analyze major U.S. macroeconomic indicators using historical data sourced from FRED (Federal Reserve Economic Data).

The project focuses on economic trend analysis, recession interpretation, monetary policy evaluation, and comparative crisis analysis using modern business intelligence techniques.

---

# 🚀 Project Overview

This dashboard analyzes key economic indicators from 2000 to present, including:

- Federal Funds Rate (FEDFUNDS)
- Consumer Price Index (CPIAUCSL)
- Inflation Rate (YoY)
- Unemployment Rate (UNRATE)
- Gross Domestic Product (GDP)

The solution was designed to support:
- Exploratory Data Analysis (EDA)
- Recession analysis
- Policy rate cycle evaluation
- Economic storytelling
- Comparative crisis analysis

# 📈 Key Features

## ✅ GDP Quarter-over-Quarter Analysis
- Dynamic GDP growth calculations
- Recession quarter identification
- Worst GDP contraction analysis

## ✅ Normalized Multi-Indicator Comparison
Implemented indexed normalization techniques to compare indicators with different scales and frequencies.

## ✅ Policy Rate Cycle Analysis
Visualized Federal Funds Rate tightening and easing cycles.

## ✅ Recession & Crisis Analysis
Compared:
- 2008 Financial Crisis
- COVID-19 Pandemic

## ✅ Event Annotation Layer
Integrated major economic events directly into dashboard visuals for improved storytelling.

# 🔍 Key Insights

- Inflation surged significantly during the post-pandemic recovery period.
- Federal Funds Rate cuts aligned closely with recessionary periods.
- COVID-19 produced one of the sharpest unemployment spikes in the dataset.
- GDP contractions were most severe during major economic disruptions.
