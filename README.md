# Macroeconomic-analysis-dashboard
Developed an interactive Power BI dashboard analyzing U.S. economic indicators including GDP, inflation, unemployment, CPI, and Federal Funds Rate trends. The project features DAX modeling, GDP QoQ analysis, recession analysis, normalized trend comparison, and economic event storytelling.

#Short summary:

Interactive Power BI dashboard analyzing U.S. macroeconomic indicators including inflation, unemployment, GDP, CPI, and Federal Funds Rate trends from 2000 to present.

 #Objectives

🔲 Analyze macroeconomic trends
🔲 Compare recession periods
🔲 Identify inflation and interest rate cycles
🔲 Evaluate COVID-19 vs 2008 crisis impacts
